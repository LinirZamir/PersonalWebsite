Research

Research Overview

Autonomous Systems such as robotic swarms are communicating on a secured protocols for achieving an agreement of operation. Such protocols are researched to improve how autonomous systems can use consensus algorithms to achieve an agreement on a set of rules in dynamic environments.

AI: Autonomous Units Consensus Algorithm

Implementation of blockchain communication protocol for autonomous units such as self-driving cars Aug 2021 - Present

  • Created a consensus algorithm designed to be implemented in autonomous systems that can verify the communicative data over a malicious environment.
  • Trust value and Digital Signature schemes among other cryptographic primitives for achieving a secured information sharing protocol.
  • Implemented with RPC written in C Programming language and Python. Simulated with ARGoS simulator.
  • Ongoing: Implement a learning algorithm to use with the new communication protocol on dynamic environment.

Localized State‑Change Consensus Algorithm

Environments (such as weather, traffic, etc.) can change frequently and unpredictably. This research offers a data validation solution for handling modular environment using consensus algorithm. MAR 2020 - Present

  • Created An algorithm designed to achieve consensus with validation of dynamic environmental data through reputation value. Assures confidentiality, integrity and validity of messages on the blockchain among all users.
  • Implemented as a sub‑ledger on the Hyperledger framework and then analyzed to ensure efficiency, scalability and security.
Collaborators:
  • Mehrdad Nojoumian, Florioda Atlantic University, Florida
Papers published:

Security Improvements of Blockchain Architecture

Improving the security aspects of known blockchain and consensus algorithms implementations MAY 2019 - JUL 2020

  • Reviewed and analyzing a large number of known and less‑known attacks on consensus algorithms and blockchain architecture.
